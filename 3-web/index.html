CSS

Types of Selectors

Simple Selectors

  • Tag/Element
    • use tags to identify the element
    • this affects all elements with that tag name
  • Class
    • Uses class attribute name to style a group of elements
    • Denoted by .className
  • Id
    • Uses a unique Id attribute to style one element
    • Denoted by #id
  • Pseudo-class
    • specific keyword at the end of the selector used to style elements in a certain state
    • IE :hover :active, etc.
  • Pseudo-Element
    • Used to style specific parts of an element
    • insert content before or after an element
    • style the first letter or line of an element
    • ::before, ::after, ::first-line, ::first-letter, etc.

Combinator Selectors

  • Child Descendant
  • Sibling
  • Attribute Selectors
  • Multiple Selectors

Wild Card Selectors

  • Just the *
  • Styles everything, all elements with the given rules
